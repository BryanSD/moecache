Welcome to memcache_client’s documentation!¶

Contents:

a minimal, pure python client for memcached, kestrel, etc.

Usage example:

import memcache
mc = memcache.Client("127.0.0.1", 11211, timeout=1, connect_timeout=5)
mc.set("some_key", "Some value")
value = mc.get("some_key")
mc.delete("another_key")

class memcache.Client(host, port, timeout=None, connect_timeout=None)[source]¶

If connect_timeout is None, timeout will be used instead (for connect and everything else)

address¶

A read-only (str, int) tuple representing the host operations are performed on

timeout¶

The timeout for reads and sends on the underlying socket (connect_timeout cannot be changed once set)

close()[source]¶

Closes the socket if its open

Sockets are automatically closed when the Client object is garbage collected
Sockets are opened the first time a command is run (such as get or set)
delete(key)[source]¶

Deletes a key/value pair from the server

get(key)[source]¶

Gets a single value from the server; returns None if there is no value

multi_get(keys)[source]¶

Takes a list of keys and returns a list of values

set(key, val, exptime=0)[source]¶

Sets a key to a value on the server with an optional exptime (0 means don’t auto-expire)

stats(additional_args=None)[source]¶

Runs a stats command on the server.

additional_args are passed verbatim to the server. See the memcached wiki for details or the spec for even more details
